记录一下来到长沙后读研阶段的吃吃喝喝。
 不出意外还有不到一年离开长沙，在毕业前将持续更新。来长沙两年来，还是吃了不少玩了不少，但一时居然想不来太多。虽然生活应重在享受而非记录，但记录仍是非常有意义的，是吧。
> 补充一下，本人精神四川人，喜辣，在CSU学习，活动范围以长沙岳麓区为中心向外辐射，评价和推荐带有极强的主观性，仅供参考。

# 一、吃喝篇
---

## 王家湾



### 大碗先生
- 地址：西二环王家湾步步高4F
- 评价：湘菜，实惠好吃，很受欢迎（感觉王家湾步步高没有它人流量要少三分之一），大桌可容纳十二人左右，包厢较少，离中南大学本部近，非常适合课题组小聚。高峰期可提前在公众号上排队取号。
- 推荐：焦糖豆腐花、血鸭、手工肉丸汤、梅菜扣肉。
- 评分： ⭐⭐⭐⭐✨

### 小吃广场
- 地址：西二环王家湾步步高广场
- 评价：很多小吃车组成道道关卡，一般人难以抗拒。各色小吃都有，起步价为8到12左右（个别小饮品只需要6块），总体偏贵，分量正常或较少，更适合开胃尝鲜或饿极了填填肚子。适合两三好友一起逛逛。和朋友说要在毕业前把各种小吃都吃遍，不知道能不能实现。
- 推荐：苕皮、狼牙土豆、铁板豆腐。
- 评分：⭐⭐⭐✨


## 麓山南路




### 新疆好地方烧烤
- 地址：麓山南路
- 评价：大份烧烤，味道不错，量比较足，环境也挺好。以前是一家兰州拉面馆子，现在好像是原班人马改成了烧烤。适合三两好友喝点beer扯扯白。三个青壮男吃撑得到情况下人均没超过100（剩了馕和羊肉火锅配菜，不过酒水是我们自己去买的强爽，这玩意儿真上头哇）。
- 推荐：红柳枝羊肉、羊肉火锅、馕（别点多了，一人一张可能吃不完）、牛油。
- 评分：⭐⭐⭐⭐
### 汉堡王
- 地址：麓山南路
- 评价：又饿又不知道吃啥就去汉堡王吧，学生优惠（打7折）四件套只要20出头，就问还有谁？不过这个优惠需要加群或者学生优惠卡，但一般只会问在不在福利群里，不会真的查验（目前为止没有被查过）。或者在淘宝上买优惠券，但不如学生优惠。但我们一致认为他家汉堡和薯条一般，但是鸡条好吃，不过也得看火候，刚出炉就很好吃。没放假的时候饭点比较火爆，适合三两好友或单人。
- 推荐：鸡条、四件套（双层芝士牛堡不要酸黄瓜+鸡块+鸡条（薯条换鸡条补2块差价+中可））
- 评分：⭐⭐⭐✨

### 北京烤鸭
- 地址：麓山南路
- 评价：奢侈一把，整个烤鸭吃吃？薄饼包上蘸酱的鸭肉片儿，再添上黄瓜丝和葱丝，一口下去，美滋滋。还可以搭配烤鸡架（肉比较少，烤的比较老）。就是薄饼粘的比较紧，戴着手套的情况下要分开稍稍要点技术。适合三两好友。
- 推荐：烤鸭片+蘸酱小碟
- 评分：⭐⭐⭐


### 原味粉馆
- 地址：麓山南路
- 评价：主打一个平价管饱，粉面盖码炒菜都有，正常价格，家常味道。盖码饭是炒一碟菜，自己打饭盛汤。他家外面还有两个冰箱，热天可以来一个雪糕边吃边等。有大桌，但不推荐，更适合三两好友。大鱼大肉吃腻了可以来吃吃家常小菜哈。
- 推荐：红萝卜炒肉、脆笋炒肉
- 评分：⭐⭐✨


### 乌龙山铁板烧
- 地址：麓山南路，有两家
- 评价：很有特色的铁板烧，小椅子，矮桌子。提供免费的瓜子吃。点好菜后炒菜阿姨会推一个小车来当面现炒，很快就可以炒好，注意别点太多了，久了会变咸。靠近中南的这一家离蜜雪冰城还贼近，点个柠檬水或者冰激凌，美滋滋。适合三五好友小酌。
- 推荐：感觉都好吃，韭菜蛋一般会必点
- 评分：⭐⭐⭐⭐✨

### 老北京涮羊肉
- 地址：麓山南路，有两家
- 评价：自助，69一位，锅底另算，鸳鸯锅20。建议三两好友在上一顿没吃或吃得少的情况下去吃个够本儿。在大冬天吃很得劲儿，很适合雨雪天或冷天吃。这时候一般人也比较多，比较热闹，烟火气满满。中南南校区北门这家东西比较全，冰激凌、蛋挞热狗、海鲜、酒水、蔬菜、甜品和水果也有一些。不过一般会猛吃现切的牛羊肉。会有阿姨不时来收盘子和加牛羊肉。关于肉的品质，我嘴没那么细，但根据我朋友的反馈，真假存疑，新鲜与否存疑。但是猛吃就完事了，一般吃完后至少半个月不会想吃牛羊肉火锅。
- 推荐：现切牛肉卷、冰激凌、刚出炉的蛋挞和热狗
- 评分：⭐⭐⭐✨
